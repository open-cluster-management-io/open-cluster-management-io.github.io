轻松接管部署在任何地方的Kubernetes集群。

Open Cluster Management 是一个由开源社区驱动的混合云多集群管理平台。在这里你可以通过简单开放的API去使用集群的注册，工作负载分发，以及动态的资源配置路由等功能。

开始

如果喜欢 Open Cluster Management，请在 GitHub 打星！

功能概览

集群注册中心

多集群元数据的注册管理以作为更上层功能的根基。

工作负载分发

通过Work API去稳步管理任意资源面向多集群的分发。

动态集群匹配及路由

动态地管理集群的拓扑以进行高级的集群匹配及路由策略。

厂商中立

避免厂商锁定问题，API通用面向厂商及终端用户可自由拓展。

在不同场景中安装管理组件

配置，保护，维护多集群资源

通过为集群打标签可以实现多集群策略配置的管理，然后平台就会帮助你维持策略配置在多集群之间的一致性，进而这些资源配置可以为你管理各个集群的访问控制，配额以及成本。

了解更多关于策略配置管理

在任何地方运行你的应用

借助应用生命周期管理相关功能，你可以创建并交付跨集群的应用，并且动态追踪应用的升级与发布。
